This is a fully working webshop built with Rails.
Features:

-Stripe Payment
-Bootstrap
-ActionCable
-Testing (TDD with RSpec and FactoryBot)
-Caching
-Optimization with Redis
-Authentication & Authorization with Devise and CanCanCan
